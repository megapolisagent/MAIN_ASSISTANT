## Активные направления

> **Маркер:** ранжировка ниже — **рабочая гипотеза MAIN_ASSISTANT** на основании текущих фактов (горизонт запасов ~2 месяца, воронка клиентов одна нить). Это **не утверждённый приоритет Марии**. Финальная последовательность фиксируется после отдельной сессии по приоритетам 60 дней.

1. **Финансы** — cashflow за 60 дней (срочно, запасы ~2 месяца, воронка одна нить)
2. **Мегаполис** — запустить публичный слой (системно, обслуживает финансовую цель через media engine → trust → leads)
3. **Здоровье** — комплексный чек-ап + зубы параллельно
4. **Дети/Родители** — систематизировать ежемесячную помощь родителям + Mode B-агент для семейной проработки (отдельно)
5. **Спорт** — питание + тренажер одной связкой параллельно

Остальные 5 сфер (Личная жизнь, Саморазвитие, Путешествия, Духовное развитие, Дом/Дача) — фоновое движение, см. [[DECISIONS/2026-06-16_wheel_of_life_baseline.md]].

## Что я сознательно НЕ делаю

- **Форекс** — до закрытия risk-test и восстановления cashflow (улика: прецедент с 6-летним долгом «в рост»)
- **Media engine как путь к деньгам за 60 дней** — цикл media → trust → leads минимум 6 месяцев, не вписывается в 2-месячный горизонт
- **Продукты для агентов** — не успевают пройти разработку до серьёзного cashflow в окне 60 дней
- **Telegram / Mobile Capture** — до результатов working loop в Computer Mode (отдельно от колеса жизни)
- **DOMAINS/** — структура — до сигнала когнитивной нагрузки ([[MAIN_ASSISTANT_CORE.md]] 2.2)
